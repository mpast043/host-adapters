# Offer - Capacity Governance Runtime

## Product Name

Capacity Governance Runtime (CGR)

## What Is Included

- CGF server (policy evaluation and decisioning)
- OpenClaw adapter
- LangGraph adapter
- Contract compliance suite
- Replay and schema lint tooling
- Workflow artifact retention and validation

## MVP Outcome

A customer can install, register a host, execute policy-governed actions, and export replayable evidence in less than one day.

## Delivery Scope

- Local or private network deployment
- Policy bundle configuration
- Host onboarding for at least one host system
- Acceptance checklist execution

## Success Metric

- Time-to-first-governed-run under 1 day
- Contract suite pass rate: 100%
- Replay and schema validation pass rate: 100%
