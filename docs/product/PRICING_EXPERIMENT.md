# Pricing Experiment (Initial)

## Goal

Identify pricing that aligns with measurable governance value.

## Experiment Arms

1. Starter
- Single host integration
- Monthly governance run cap
- Basic replay retention

2. Team
- Multi-host integration
- Expanded run volume
- Enhanced retention and validation workflows

3. Enterprise Pilot
- Custom policy bundle support
- Dedicated onboarding and hardening
- Priority support for deployment blockers

## Evaluation Metrics

- Conversion from pilot to paid
- Time to first successful governed workflow
- Monthly active governed runs
- Contract compliance retention over time

## Notes

Do not set public prices until 3 paid pilot conversations are completed.
